# LingCode 资源目录

本目录存放输入法运行所需的各种资源文件，主要从 Rime 项目中引用。

## 目录结构

- `schemas/` - 输入方案配置文件 (来自 rime-prelude、rime-luna-pinyin 等)
- `dicts/` - 词典文件 (来自 rime-luna-pinyin 等)
- `opencc/` - OpenCC 简繁转换配置 (来自 rime-opencc)

## 资源来源

资源文件主要来自以下 Rime 项目仓库：
- [rime-prelude](https://github.com/rime/rime-prelude) - 基础配置和默认设置
- [rime-luna-pinyin](https://github.com/rime/rime-luna-pinyin) - 朙月拼音词典和方案
- [rime-double-pinyin](https://github.com/rime/rime-double-pinyin) - 双拼方案
- [rime-opencc](https://github.com/rime/rime-opencc) - OpenCC 简繁转换数据

## 使用说明

1. 这些资源文件遵循 Rime 的开放协议
2. 在使用时需要保留原作者的版权信息
3. 具体的资源文件可以根据需要从上述仓库下载
