# 灵码输入法

## 任务目标
实现一个跨平台中文输入法框架，兼容 Rime 生态，支持拼音和双拼输入

## 方向协议（每次会话开始执行）
1. 阅读 CHANGELOG.md 了解当前状态
2. 查看"下一步任务"列表
3. 选择最高优先级任务
4. 确认理解任务范围后再开始编码

## 成功标准

### 第一阶段：核心基础（Phase 1）
- [ ] 定义核心数据类型（InputState、KeyEvent、Candidate）
- [ ] 实现输入引擎状态机（Idle -> Composing -> Selecting）
- [ ] 实现基础拼音匹配器（全拼转汉字）
- [ ] CLI demo 能正常输入中文
- [ ] 核心库单元测试通过
